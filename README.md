# 📁 Auto File Organizer

Pythonで開発したデスクトップアプリです。

指定したフォルダ内のファイルを拡張子ごとに自動整理できます。

---

## スクリーンショット

(画像)

---

## 主な機能

✅ フォルダ選択

✅ ファイル自動整理

✅ カテゴリ別に分類

✅ ログ表示

✅ 整理件数表示

---

## 使用技術

Python

Tkinter

pathlib

shutil

Git / GitHub

---

## フォルダ整理例

Before

Downloads
├── cat.jpg
├── report.pdf
├── music.mp3

↓

After

Downloads
├── 画像
├── PDF
├── 音楽

pip install -r requirements.txt

python main.py
